# 🧊 LLM-Based 3D Model Generator

A web application that generates 3D models from text descriptions and images using Large Language Models.

## 🌐 Live Demo
- **Frontend:** https://frontend-mocha-three-73.vercel.app
- **Backend:** https://3d-model-generator-production.up.railway.app

## ✨ Features
- 📐 Text to 3D model generation
- 🧠 Image to 3D model (Groq Vision)
- 🔧 Refine models (bigger, taller, smaller)
- ⬇️ STL download for 3D printing

## 🛠️ Tech Stack
- Frontend: Next.js, TailwindCSS, Vercel
- Backend: Python, FastAPI, Railway
- LLM: Groq API (Llama 3.1)
- 3D Engine: OpenSCAD
